American Legion Jersey Boys State Schedule Project

Client

  • American Legion Jersey Boys State

Purpose

  • Edit and redesign multiple versions of weekly schedule

Notes

  • Edited and redesigned two schedules and two manuals
  • Created responsive webpage version of staff schedule
  • Created consistent graphic design for documents

For my capstone project in the Technical Communication program, I decided to redo the ALJBS delegate and staff schedules. This was an ambitious project with multiple phases and it got even bigger than I first anticipated.

First, I did a thorough copy edit of the current schedules, trying to tighten the wording without harming it as much as possible. Next, I created a new layout for the schedule, making it easier to scan quickly. Next, I reorganized the staff manual so that it would be easier to locate information. Finally, I created a webpage for the staff schedule, incorporating a simple responsive design so that it would be viewable on staff members' smartphones and tablets.

This was a huge, difficult project (especially since I had no experience with responsive design) but everything turned out extremely well.
