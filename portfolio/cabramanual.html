Cabra User Manual

Client

  • Cabra (hathix.com)

Purpose

  • Create user manual for free flashcard program

Notes

  • Written in Microsoft Style
  • Manual was amended after a major software update two weeks before the deadline
  • Created in Microsoft Word
  • Designed with lots of color due to middle-school audience

Cabra is a free flashcard program aimed largely at children aged 6-12. Neel Mehta, who wrote the program, has a unique, quirky voice and wrote the program with a mixture of encouragement and humor. Because of this, the manual had to keep Neel's voice and not be boring in addition to the usual standards. I used a lot of color (and pushed Microsoft Word's layout features to the limit) when creating this, trying to design something that the users would enjoy.

Note: because .docx format was a requirement, .pdf output was not considered. A .pdf version is used here to alleviate any font issues but suffers from a loss of quality. The original document is available upon request.
