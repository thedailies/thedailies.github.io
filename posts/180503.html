Link of the Day

Pure CSS Francine

CSS is the thing that's responsible for making websites have color and be attractive to look at. Diana Smith, a front-end developer, uses it to make hand-coded "paintings."> Her three rules:

1. All elements must be typed out by hand
2. Only Atom text editor and Chrome Developer Tools allowed.
3. SVG use is limited, and all shapes can only use hand-plotted coordinates and bezier curves - without the aid of any graphics editor.

This is bonkers crazy. The Simpsons in CSS was hard enough. This is a completely different level of [self-imposed] difficulty.

If you want to see how Smith did it, you can view her code over on Github.
